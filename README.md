# Shop SQL + JS
# Built by
Sudip Debnath

**Features**
- Lightweight Node.js scripts to manage a shop-like SQLite/SQL workflow.
- Simple files: `database.js`, `insert.js`, `display.js`, `delete.js`, `tables.js`, `specific.js`.

**Quick Start**
1. Install dependencies:

```bash
npm install
```

2. Run a script (example):

```bash
node specific.js
```

**Project Layout**
- `database.js` — DB connection helpers
- `tables.js` — schema creation
- `insert.js` — insert sample data
- `display.js` — list rows
- `delete.js` — remove rows
- `specific.js` — example runner for a specific flow

**Tips & Notes**
- Keep secrets out of the repo — add them to an `.env` file (already ignored).
- Use `Outputs/` for generated/exported artifacts (ignored by git).

**Contributing**
Feel free to open issues or create PRs. Keep changes focused and well-tested.
